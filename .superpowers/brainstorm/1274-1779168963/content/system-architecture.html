Pipeline (same flow as stalls)

① Scrape — Firecrawl fetches the client's existing site (text + images)
② Generate — Claude produces sector-specific JSON (brand, services/menu/team, colors, copy)
③ Images — Gemini generates → Firecrawl scraped fallback → Unsplash keyword fallback
④ Review — Sales rep previews brand + images before committing
⑤ Deploy — Writes config files + images to GitHub → Vercel auto-deploys → preview URL returned
